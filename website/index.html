Git - zakładanie repo.

Zakładanie repo lokalnie i push do repo zdalnego.
  1. git config --global user.name "tbl4h"
  2. git config --global user.email przykładowy@email.com
  3. git init // wewnatrz katalogu naszego repo
  4. git add .
    • git rm nazwa_pliku
  5. git status // sprawdzenie czy wszystko ok
  6. git commit -m "przykładowa_tresc"
  7. git remote add origin https://github.com/tbl4h/nazwa_repo.git
    // repo zdalne musi byc utworzone i miec tą samą nazwę.
    • git remote remove origin
  8. git push -u origin master
Na koniec jeszcze link do podrecznika gita.
